# Introduction
## What?
The FBR app is an interactive datadex. 

Meaning it works like a pokedex for coffee, forming its large network/interactive library from user-generated data. This datadex will be used to optimise current and any future FBR tech eg. mDot and the dial. We want to motivate people to use and explore the platform.

## Where?
The FBR App is a web application embedded in the FBRdex. It will be used across desktop, tablet, and phone devices.
