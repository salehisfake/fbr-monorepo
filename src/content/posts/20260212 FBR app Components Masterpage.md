# FBR App Components Masterpage

Since components appear all over diff. sections and pages, this  serves as a general guideline or an index to almost all possible app components.

App components are all the smaller things that make up the page for example, a profile snapshot, a tag, a post, a navigation bar. Below, you will find that some components have already been identified and sectioned away in the top left corner. 

<iframe style="border: 1px solid rgba(0, 0, 0, 0.1);" width="800" height="450" src="https://embed.figma.com/design/kCbuZ9kfpuhTsUUxnmaddI/FBR-App-interface?node-id=2-2&embed-host=share" allowfullscreen></iframe>

Where appropriate, there are 3 views of components:
- what it would look like **condensed** eg. a condensed profile could just be the user profile picture and username
- what it would look like **normally** eg. a profile view could normally look like a user profile, username, and contribution graph
- what it would look like **expanded** eg. an expanded profile view could include user profile, username, contribution graph, user bio, the user's links to their other social platforms
